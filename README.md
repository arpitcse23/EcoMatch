# 🌱 EcoMatch – AI-Powered Circular Campus Reuse Platform

EcoMatch is a full-stack climate-tech platform designed to reduce campus waste by intelligently matching unused items with students who need them — while tracking measurable environmental impact.

---

## 🚨 Problem

University campuses generate large amounts of waste due to:
- Unused electronics and furniture
- End-of-semester disposal
- Lack of structured reuse systems
- No measurable sustainability tracking

There is no intelligent reuse ecosystem.

---

## 💡 Our Solution

EcoMatch creates a smart circular reuse network with three core modules:

### 🔹 1. Smart Listing Hub
- AI-assisted item categorization
- Condition tagging
- Location-based listing
- Intelligent suggestions

### 🔹 2. AI Intelligence Hub
- Weighted match scoring (proximity + category + urgency)
- Claim likelihood forecasting
- Smart reuse insights
- Demand trend visualization

### 🔹 3. Green Impact Hub
- CO₂ saved tracking
- Waste prevented tracking
- Eco score & green points
- Hostel leaderboard system
- Live campus sustainability pulse

---

## 🧠 How It Works

- Users list unused items
- The system calculates intelligent match scores
- Other users claim items
- The platform updates environmental impact metrics in real time
- Hostels compete via sustainability leaderboards

## 🛠 Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- SQLite

---

## ⚙️ How to Run Locally

### Backend
```bash
cd Backend
uvicorn main:app --reload
```

### Frontend
```bash
cd Frontend
npm install
npm run dev
```
